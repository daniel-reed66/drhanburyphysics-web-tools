## What drhanburyphysics is

drhanburyphysics is a collection of web tools built around physics education for A level and IB learners. The aim is to present core ideas through interactive content that works well in lessons, revision periods, and guided study sessions.

It is a good fit for teachers who need ready-to-use digital material and for students who prefer an interactive way to revisit physics concepts. Since everything runs in the browser, it avoids a local install process and is straightforward to use in shared or classroom settings.
